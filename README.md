# Tucil1_13524109

## Deskripsi Program
Program merupakan Queens Solver, yakni program untuk menyelesaikan persoalan Queens, sebuah papan permainan berbentuk persegi. Persoalan ini memiliki aturan untuk menempatkan queens tidak dalam satu kolom dan baris yang sama. Selain itu, hanya boleh terdapat satu queen di satu daerah warna yang sama. Posisi queens juga tidak boleh berdekatan atau bertetangga (jarak kotak = 1).

## Requirement
- Python 3.10
- PyQt6 (untuk GUI)

## Command untuk Install PyQt6
pip install PyQt6

## Command untuk Menjalankan Program
- GUI: cd src lalu python gui.py
- CLI: cd src lalu python main.py

## Keterangan untuk Output
Jika memilih opsi "Save Result .txt" pada GUI, maka solusi akan disimpan ke dalam dokumen baru, tanpa menimpa dokumen masukan.

## Keterangan untuk bin
Untuk bahasa Python, program tidak perlu dikompilasi.
